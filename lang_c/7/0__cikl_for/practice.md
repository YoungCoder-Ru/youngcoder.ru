**2\.** Листинг 20 можно было бы переписать следующим образом: 
```c
#include <stdio.h>

int main(void)
{
        for (int i = 1; i <= 10; i = i + 1) {
                for (int j = 1; j <= 10; j = j + 1, printf("\n")) {
                        printf("-%4d", i * j);
                }
        }

        return 0;
}
```

Здесь вызов функции `printf("\n")` перенесён в третье выражение из заголовка цикла. Для этого используется оператор `,`. Разберитесь самостоятельно, как ещё его можно использовать в заголовке цикла `for`. 

В комментариях к уроку можете переписать программы из этого урока с использованием `,` в заголовке цикла `for`.

Обратите внимание, что хотя третий блок обычно используется для изменения значений параметров цикла, мы всё же имеем право в этом блоке делать и другие вещи.
