## Дополнительные материалы

**1.** Используя циклические конструкции `while` и `do-while` тоже можно создать бесконечный цикл. Хотя в этих конструкциях нельзя просто оставить пустыми скобки с управляющим выражением (как в цикле `for`), но можно записать туда, например, значение `1` и тогда цикл получится бесконечным.
 
```c
// эти варианты не сработают.
while () {
        printf("Infinity loop!\n");
}

do
        printf("Infinity loop!\n");
while (); 

// а так уже будет работать
while (1) {
        printf("Infinity loop!\n");
}

do
        printf("Infinity loop!\n");
while (1); 
```


**2.** Если вам нравятся сериалы на хакерскую тематику, то посмотрите сериал Mr. Robot. 

Главного героя сериала зовут Elliot Alderson. Из сериала можно узнать, что существует отдельный "вид" бесконечных циклов, которые называют [Alderson loop](http://www.catb.org/jargon/html/A/Alderson-loop.html). Термин совершенно не распространённый, и даже из английской википедии удалили упоминание о нём ([хотя раньше оно там было](https://en.wikipedia.org/wiki/Talk%3AInfinite_loop#%22Alderson_Loop%22_example_is_incorrect)). Отдавая дань уважения сериалу, я решил о нём упомянуть. 

В общем, =Alderson loop= --  это такой бесконечный цикл, в котором есть условие выхода, но из-за ошибки в коде программы это условие не может быть выполнено.

Пример такого цикла на основе нашей программы «Игральный кубик»:
```c
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void)
{
        srand(time(NULL));

        printf("#########  Devil\'s bones   #########\n");
        printf("#                                   #\n");
        printf("#   Commands:                       #\n");
        printf("#                                   #\n");
        printf("#   1 - new game                    #\n");
        printf("#   0 - exit                        #\n");
        printf("#                                   #\n");
        printf("#####################################\n\n");

        int control;
        int value = 0;
        
        do {
                printf("Enter command: ");
                scanf("%d", &control);
                
                if (control = 0) { // ОШИБКА! Нужно ==, а не =
                        printf("Good bye!\n");
                        return 0;
                }
                
                switch (control) {
                        case 1:
                                value = 1 + rand() % 6;
                                printf("Result: %d\n", value);
                                break;
                        default:
                                printf("Error! Try again...\n");
                                break;
                }

        } while (1); 

}
```

Идея этого кода была в том, чтобы избавиться от дублирования кода за счёт использования бесконечного цикла `do-while`, внутри которого использовать подход early return, но из-за ошибки в записи условия мы получаем, что управляющее выражение в `if` всегда `ЛОЖЬ`, а значит мы никогда не сможем завершить программу. Стоит исправить `=` на `==`, как программа начнёт работать корректно.
